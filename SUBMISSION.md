## Project Overview

This project is a full-stack Loan Management System that allows borrowers to apply for loans and enables internal teams to manage the complete loan lifecycle from application to closure.

The system includes:

* Borrower Registration & Authentication
* Business Rule Engine (BRE) Validation
* Salary Slip Upload using Cloudinary
* Loan Application Workflow
* Loan Approval & Rejection Workflow
* Loan Disbursement Workflow
* Repayment Collection Workflow
* Automatic Loan Closure
* Role-Based Access Control (RBAC)
* JWT Authentication
* bcrypt Password Hashing
* Dedicated Dashboards for All Roles
